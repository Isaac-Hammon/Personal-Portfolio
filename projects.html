Additional Projects

Game of War Project

A console-based “Game of War” application built using C#. The project simulates a two-player card game where each player draws from a shuffled deck and compares card values to determine the winner of each round. It demonstrates core programming concepts such as loops, conditionals, object-oriented design, and basic game logic implementation.

Weather Checker

An app that I created where you enter a city name and it fetches the current weather using a public API (like OpenWeatherMap). Displays temperature, humidity, and weather conditions. Skills Practiced: API requests, JSON parsing, asynchronous programming, and working with APIs.

Digital Journal

A personal diary app where users can write, edit, and delete entries. Data is stored locally on the device. Skills Practiced: File handling or local storage, CRUD operations, and UI design.
